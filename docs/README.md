## Project documentation

Use this folder to store **design documents, reports, and diagrams** related to the bitonic sorter project, for example:

- PDF reports (e.g. architecture/RTL descriptions)
- Block diagrams or timing diagrams
- Lab or project write‑ups

At the moment, there is at least one PDF in the project root:

- `rtl view sort 3.pdf`

For a clean structure, move that file (and any similar documents) into this `docs/` folder so that:

- Source code lives in `src/`
- Quartus project files stay at the project root
- Documentation is grouped here in `docs/`
